# Quantum Key Distribution (BB84)
## Description

--> This project demonstrates the BB84 Quantum Key Distribution protocol using Python and Qiskit to generate a secure shared key between Alice and Bob.

-->Quantum circuits are used to encode, measure, and compare bases, producing a secret key when both parties choose matching bases.

## Tools Used
- Python
- Qiskit
- Google Colab

## Output
The program generates a shared secret key between Alice and Bob when both use matching measurement bases.
