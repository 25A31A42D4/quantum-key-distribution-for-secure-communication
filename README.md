# quantum-key-distribution-for-secure-communication
This project simulates the BB84 Quantum Key Distribution protocol using Python and Qiskit to generate a secure shared key between Alice and Bob. Quantum circuits are used to encode, measure, and compare bases, producing a secret key when both parties choose matching bases.
# Quantum Key Distribution (BB84)

## Description
This project demonstrates the BB84 Quantum Key Distribution protocol using Python and Qiskit. 
It simulates secure key exchange between two parties, Alice and Bob, using quantum circuits.

## Tools Used
- Python
- Qiskit
- Google Colab

## Output
The program generates a shared secret key between Alice and Bob when both use matching measurement bases.
